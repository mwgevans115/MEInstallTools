# Get-DownloadFolder

## SYNOPSIS
Function to return the download folder

## SYNTAX

```
Get-DownloadFolder [<CommonParameters>]
```

## DESCRIPTION
Function to return the download folder since it is not a standard well known folder

## EXAMPLES

### EXAMPLE 1
```
Get-DownloadFolder
```
