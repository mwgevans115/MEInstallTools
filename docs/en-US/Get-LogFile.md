# Get-LogFile

## SYNOPSIS
Function to return the default log file for a script based on
BasePath\CompanyName\ProductName\ProductVersion or
if the script has a LogPath parameter it will use that

## SYNTAX

```
Get-LogFile [<CommonParameters>]
```
